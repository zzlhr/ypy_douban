# ypy_douban

### 多线程豆瓣图片爬虫

![](https://ws1.sinaimg.cn/large/005tjobOly1ft8a9nzkftg30vs0nkqv5.gif)
```bash
git clone https://github.com/zzlhr/ypy_douban.git

cd ypy_douban

# 创建存放图片文件夹
mkdir mz

python ./main.py

```

修改线程数和关键字

```bash
# vim main.py and update params

# 线程数
thread_num = 10
# 关键字
keyword = '妹子'
```
